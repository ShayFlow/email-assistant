# Email Assistant

This project drafts email replies in the user's personal communication style.

## How to use
Paste an incoming email. Follow `workflows/reply_to_email.md` and return a structured response with 5 labeled sections: intent, missing info, subject, draft reply, style note.

## Critical rules
- Read `style_guide.md` before drafting every reply
- The #1 quality bar: the reply must not sound AI-generated
- Never invent facts or make promises
- Flag missing information explicitly
- Tone: practical, polite, brief, human

`workflows/reply_to_email.md`

## Style reference
`style_guide.md`
